# Zápočet
- Vytvoření praktické seminární práci
    - Vymišlení tématu (na příklad: Knihovna, Pokemon, ... )
    - Navržení
    - Implementace
    - Prezentace
## Požadavky
- Navrhnout databázi s minimálně 7mi tabulkami a propojeními
    - ERD diagram (Konceptualní návrh)
    - Implementace v postgreSQL sqlite
    - Alespoň 20 záznamů v každé tabulce v průměru (alespoň 200/300 záznamů celkem)
    - Realná data
    - Použití příkazu 
        - Select
        - From
        - Where
        - Group
        - Having
        - Order by
        - Vytvoření virtualní tabulky
            - Create view as select
            - Spojení tabulek alespon 2-ma joiny
                - Join
                - Left join
                - Right join
                - Full join
