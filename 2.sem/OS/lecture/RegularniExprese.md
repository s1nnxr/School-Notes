# Druhy regulárních výrazů (Regular Expressions)
- zakladní regulární výrazy-BRE
    - Pro BRE jsou speciální znaky: . [ ] * ^ $ \
- rozšířené regulární výrazy-ERE
    - Pro ERE jsou speciální znaky: . [ ] * ^ $ \ ( ) { } ? + |
- Perl-kompatibilní regulární výrazy-PRE
    - Pro PRE jsou speciální znaky: . [ ] * ^ $ \ ( ) { } ? + | < > = ! : &

# Nástroje
- grep, egrep (grep -E) -BRE,ERE, PRE-experimentálně 
- C#- třída Regex 
- Java - třída Pattern 
- C, C++ - knihovna PCRE
